前言: 2022年夏天我在该地调研，2024年阅读黄宗智的《长江三角洲的小农家庭与乡村发展》后所写。

摘要: 由于恶劣地理条件限制，陕西省安康市在改革开放以前都未摆脱“马尔萨斯陷阱”。1978年改革开放后，东部沿海地区的经济活力向内地传导，安康市经济才迎来快速发展。因而当地经济发展依赖于外来技术与产业的引进，在安康市内部，经济发展水平在地理上呈现出从平地到山地、从低海拔到高海拔的渐进发展模式。

秦巴山区的生态系统 #

文章的研究地区是陕西省安康市。该地北依秦岭，南接巴山，汉江于两山之间川流而下，其地貌呈现南北高山夹峙、 河谷盆地居中的特点，人类活动多位于汉水附近的山间谷地。在全区国土面积中，大巴山约占60％，秦岭约占40％；山地约占92.5％，丘陵约占5.7％，川道平坝占1.8％。根据海拔分类：1800米以上的高海拔地区占国土面积的10%，800-1800米的中海拔地区占国土面积的55%，170-800米的低海拔地区占国土面积的31%。

图1：安康市行政区划图

上图标明了该地的位置，该地区虽属于陕西省，但在气候特征上却偏向于南方特征（但农业耕地仍然以旱地为主，兼有部分水田）。气候为亚热带大陆性季风气候，气候湿润温和，四季分明，雨量充沛，无霜期长，年平均降水量为938毫米 。

由下表可知农地与林地加起来已经超过了90%（其中又以林业用地为主），居民用地与工业用地规模有限。

人口 #

安康市的人口分布非常不均衡：安康地区海拔500米以下的川道丘陵区，土地面积约占全区总面积的3％，人口却占20％以上；海拔500—800米的低山区，土地面积占31％，人口约占60％；海拔800—1800米的中山区，土地面积占55％，而人口总数不到全区的20％（但如果考虑到人口基数，这一数字也是庞大的）。

建国后，该地人口迅速增加，但到1990年，农业人口数量都保持在90%左右，1990年到2010年间这一数字已经有所变化，降低为83.97%，但一直持续到2015年的统计口径变化前，这一比例都基本上保持稳定，变化较为缓慢。2015年之后，城镇化率大幅度提升，直到2020年，城镇化率已经接近50%。因而文章在时间节点上分为2015年之前与2015年之后两段，分别进行说明。

1949——1979 #

建国前，该地经济结构较为单一，几乎没有近代化的工业工厂。虽然城镇出现了，但由于狭小地理环境的限制，城镇人口数量较少，面积也不大。并且城镇洪涝灾害不时威胁着城市居民，给城市带来灭顶之灾（直到20世纪80年代这种情况仍然很严重，如1983年汉水泛滥，冲毁安康老城房屋1.16万间，死亡870人）。近90%的人口仍零散分布在广袤的山区中，形成了一个个自然村。由于大山阻隔，这些村庄之间的往来交流极其不方便。

从上表中可以看出该地经济上以农业与副业（手工业）为主。由于“八山一水一分田”的地理格局，在有限的耕地上，农民主要种植农业作物，以求温饱，但尽管如此，由于自然环境的恶劣，该地居民的生活水准仍然远低于长江三角洲与华北。直到1985年，据省委调查组统计，全区仍有96.53万人的温饱问题尚未解决，占全区农业人口的37.7% 。

副业以山货特产的形式表现。特产主要有蚕桑、茶叶、生漆、油桐、药材这几类，并且其中一些产品已经达到了很高的商品化水平（但有待于进一步考察）。城镇是这些山货的加工地与流转地，这些副业是农民收入的主要来源。

当地农业耕地具有以下的几个特点：

  1. 坡地多，平地少。
  2. 海拔高，土层薄。
  3. 零星碎片分布，宜农耕地面积少。

农作物以粮食作物为主，辅以棉花与油菜等经济作物。其中粮食作物年播种面积占农作物面积的90％左右，产值占农作物的70％以上，分夏秋两季收获。玉米地位突出，常年播种面积占秋粮的一半以上，产量占全年粮食总产的三分之一。 如果说“内卷”使得长江地区人口的收入处于温饱线附近，那么安康地区则仍有大量居民未达到温饱线。因此建国后，增加粮食产量，实现温饱就成了该地政府的首要目标。

蚕桑 #

该地桑蚕业历史很悠久，地位显著。蚕桑业早在明清就有很大的发展，当地官员也重视该产业发展，鼓励当地民众从事该行业，并学习农书，提高技术。桑蚕业促进了城镇的繁荣，民国时期安康县城关、王彪店、洋溢河，汉阴县城关、涧池、蒲溪和石泉县池河一带，遍布丝织作坊。民国20年(1931)，全区蚕茧总产达到121.5万公斤，占全省产量的73.4％，为建国前安康地区年产茧最高水平。据部分业者回忆：王彪店、洋溢河一带的丝织作坊，拥有单机者千余户，拥有3至5台织机者、百余户，拥有5至7台织机者50余户。

抗战及其随之而来的解放战争，对该地蚕桑业产生了极大冲击。至建国前的1949年，全区桑园面积仅存1.1万亩，生产蚕茧20.76万千克，只占抗战前最高年产量的17.9％。建国后，蚕桑业经过恢复迅猛发展（虽然文革与大跃进不可避免地对该行业造成了破坏）。

建国前，安康地区桑树多在房前屋后地边、路边、坎边和渠边零散栽植，成片栽植很少。建国后，多为集体成片栽植。到70年代中期，开展大规模植桑会战，利用荒山、荒坡、荒滩、开带修坎，成片成带植桑。

茶叶 #

茶业与桑蚕业在某种程度上具有相似性。由于合适的要素禀赋，该地同样自古以来就有生产茶叶的历史。古代的“茶马贸易”为安康地区开辟了可靠稳定的市场。自清末到1949年，全区茶叶的收购和销售全为私商。抗日战争前，茶叶价格平稳，又带有较强的季节性，经营茶叶有利可图，所以茶商比比皆是，尤以回族商人最多。县城成为全县茶叶加工、经营、转销地。茶商在安康以低价买进茶叶，雇工在城镇进行加工后转销至西安、或由水路到汉口高价卖出，以获得利润。

建国前茶园主要由私人经营。建国后，出现国营、集体和个体三种形式的经营体制。茶叶初制由茶农在家手工进行，建国后通过引进机械、建立茶厂。1954年茶叶经营已基本上由国家控制，统购统销，1955年国家严禁私商进入产茶区经营茶叶，1956年国家对私营工商业进行社会主义改造后，茶叶列为二类物资，完全由国家统购统销，1984年之后，实行议购议销，由市场调节，产销见面，国家、集体、个体竞相经营，促进了茶叶的流通和生产的发展。其中该地紫阳县是全国第二高硒区，为安康地区特有的富硒茶区。

生漆 #

漆树是特用经济树种，民间自古有“漆树产八料”之说，即涂料、油料、饲料、木料、糖料、肥料、药料、燃料，在工业、农业、国防和人民生活中有广泛的用途。明末清初该地生漆业有较大发展，生漆经汉口运往江、浙、湖、广，再转销日本、东南亚诸国。安康地区生漆产量跃居全国之首，占全国总产量的20％，全省产量的70％，出口量占全国的25％。岚皋、平利两县产量连续多年名列全国产漆县的第一、第二名。

建国初期，漆树分配给私人或归国家，生漆大部分谁采谁有。合作化时期，漆树随地入社，人民公社化时，漆树归生产队经营，1981年落实林权归属，漆树折价到户，实行国家、集体、个人一起上及谁造谁有的政策。

民国初，仅平利县就有漆商70余家，规模大的有饶逊安、李次责、康品三等。三四十年代外商纷纷利用买办商人采购生漆。建国后，生漆由国营商业部门经销，大、小商贩渐被取缔。直到中共十一届三中全会后，方出现个体生漆贩运、经营者。

油桐 #

第一次世界大战后，安康桐油在汉口成为日、美、德商人争购的主要产品，价格随之上扬，桐油在安康得到迅速发展。1936年和1937年安康地区桐油产量达936万公斤。安康城关桐油商行多达30多家，为安康地区桐油发展鼎盛时期。抗战爆发后，桐油业衰退。

民国时期，油坊主到产桐区农民家里采取预付货款、赊销货物等办法收购。建国初期，主要靠私商收购，公私合营后，才把桐籽定为二类统购统销产品，以粮食部门经营为主，实行统一收购，计划调拨，严禁私人倒卖贩运。

结论 #

通过对以上安康地区的主要产业进行考察，我们可以得到如下的一幅图景：

1949年前：安康地区的存在一定规模的农业与副业，农业是人口发展存活的条件；而副业则是该地区增收的重要依赖。并且副业已经达到了较高的商品化水平，各地商人在安康低价买来丝绸、茶叶、药材等产品，再到异地高价卖出以此而谋利。抗战爆发后汉口沦陷，对当地部分产业造成了极大的打击。

1949-1979：新中国成立后，农业技术的发展，粮食生产确实有上升。副业在此期间也从战争中逐渐恢复而增长。在这一时期实行的是“计划经济”，统购统销的实施，垄断了贸易。

1979年改革开放之时，安康地区仍然非常贫穷。1973年，全区平均每个劳动日值仅0.35元，年人均分配现金收入47.03元、口粮177公斤。

1979-2015 #

1979年，当文化大革命结束，国家将中心任务重新回归到经济建设上时。安康仍然是一个贫穷的地区，大部分居民仍然生活在乡村，基础设施建设不足，但情况仍然比建国之前要好一些。经过30年来的建设，安康市的农业技术已经有所增长，并且有了初步的集体与国有工业。服务业也发展起来，二者加起来已经占到了GDP比重的近一半。更为重要的是，有了稳定的政治与经济环境，居民可以与政府一起同贫穷与自然灾害作斗争。

多种经营与城乡差距 #

在这一阶段政府的各项文件与会议中，“多种经营”成为了频繁出现的热词。1978年安康市委专门成立了“多种经营办公室”（正局级单位），辖区内各县也先后成立了“多种经营办公室”。1979年市委会议确立“争取粮食自给有余，并且发展多种经营”的目标。具体来说，是在“八山一水一分田”的格局下，依靠一分田来满足温饱；依靠“八山”来发展多项经营。

1985年林业区划调查，全区林业用地2456.5万亩，占全区土地总面积的71％；其中有林地1373.2万亩，占林业用地的57％，森林覆盖率为39.2％。林特产值(不含群众自用材、烧柴、食用菌、果等)，约占当年农业总产值的12.19％。

表13：1978年以来安康城乡居民人均可支配收入统计

由上表可知，1979年后农村与城镇居民收入都得到发展。但随之而来的是城乡差距的扩大。城镇与乡村收入差距一度达到3到4倍。考虑到大部分居民仍然生活在农村，这从侧面反映：

  1. 改革开放后对经济松绑，但城市增收手段更多。

  2. 只有少部分城镇居民获利，经济发展的福利并未深入到广大山区农村。

乡镇企业 #

乡镇企业是实行多种经营的载体之一。改革开放后，生产责任制度逐步建立起来。各大队对辖区内一部分农业与林业资源承包给了个体进行经营。在一些行业由于集体劳动的需要，集体化时期的一些集体企业则发展成为乡镇上的社队企业。这些社队企业涵盖以农业及其对应的轻工业（如纺纱、织布）为主，也经营采矿、机械等部分重工业。这些乡镇企业是农民收入的重要源泉。

表14：1949年以来安康农林牧渔产值统计

至1983年，全区2.3l万个生产队中，实行家庭联产承包责任制的占99.83％，其中实行“大包干”的占总队数的98.57％。集体经济力量雄厚、经济收益较高，分包到户不利于发展生产的大队和生产队，在群众自愿的基础上仍然实行统一经营，统一核算，统一分配，只在生产过程中实行其他形式的责任制。

1989年， 全区多种经营生产虽然还比较落后， 但它在农村经济中占重要地位。 全区多种经营产值占农业总产值一半以上， 达59.8%； 农民人均纯收入299.5元， 而多种经营产值占171元， 多种经营在农业中的支柱地位基本确立。同一年开始第一轮农业科技承包，科技承包较好地解决了土地承包到户如何把农业技术送到千家万户的问题， 加速了农业科学技术向生产力转化的进程。

在这一时期，该地农村经济发展有两个主要方向。第一是在传统产业如蚕桑、油料方面继续深入发展，创造就业机会，增加产值。另一方面，该地也探索出了一些新的行业。在农业方面农民种植烟草，该地逐渐建立起完整的烟草产业链；工业方面化学工业、酿酒等也发展起来。

1997年桑、茶、烟、果、药等主导产业开发初具规模，一批“龙头”企业和服务体系相继建立，“贸、工、农”一体化、产加销“一条龙”的生产经营新格局正在逐步形成。多种经营生产成为覆盖乡镇、村、户温饱“越线”的主导产业。农业多种经营的发展，不仅增加了农民收入，而且也使地方财政收入不断增长。

移民搬迁 #

20世纪90年代，温饱问题在当地逐渐得到解决。扶贫开发的目标由以解决剩余贫困人口温饱问题为主， 转入以解决相对贫困问题为主的新阶段。当时安康地区70%贫困人口居住在地理位置偏僻、 社会发育程度较低的中高山区。自然环境限制了产业的发展。

1997年，安康地区开始实施移民搬迁扶贫安置点建设，时称“异地扶贫搬迁”。这种搬迁部分是出自于扶贫，保障生命安全也占了很重要的因素。安康地区夏季暴雨频发，经常导致山间泥石流等自然灾害。移民搬迁最主要的目的就是将人口从高海拔不宜居的地点搬迁到低海拔相对宜居地区。

1998年政府印发移民搬迁相关方案。规定对移民进行补助，补助标准： 以县为单位， 按每移民搬迁1个贫困人口， 补助移民搬迁费1000元。要求地、县各配套补助500元。1997到2005年，安康市迁移安置贫困群众近12万人。仍有6.17万户24.68万贫困人口居住在条件较差的高寒边远山区，需要移民搬迁扶贫。

2010年可以作为移民搬迁行动的分水岭。2010年，安康遭受“7 • 18”特大洪涝泥石流灾害，造成安康10个县（区）168个乡镇受灾，受灾人口155万人， 紧急转移安置受灾群众58万人， 直接经济损失65.6亿元。

这次灾情引起了全国范围内的注意。受灾区域也超出了安康一地范围，陕南的汉中与商洛二市同样是严重的受灾区。因而陕西省政府开始在全省范围内推进“移民搬迁”的战略。安康市依据陕西省省政府“总体规划” 的要求， 全市结合实际， 规划搬迁22.6万户88万人。

农村基础设施建设 #

截至2015年，该地扶贫可以分为2个阶段。

1979—1997：以解决温饱，同时多种经营创收为目的。

1997—2015：以村庄建设为主体，整村推进扶贫。

我认为该地的经济发展呈现一种扩散型，即由城镇—低海拔乡村—中高海拔乡村的这样一种扩散趋势。改革开放后东部沿海地区经济首先爆发出巨大活力，慢慢传导到内陆的安康地区。在乡镇企业兴起，各种现代技术逐渐进入当地时，城市由于地理条件最为优越，最容易接触到这些经济中的活力因素，因而城市居民的收入出现了飞速增长。这一增长也带动了一些地理位置较好的乡村的发展，但由于粮食作物与经济作物之间的竞争，以及乡村相较于城市较差的地理环境，由副业与工业所支撑的经济发展较为缓慢。至于位置更偏僻的一些乡村（如居住海拔800米以上），由于环境的闭塞而与外界隔绝，很少分享到经济发展的活力。这也解释了为什么直到2015年各地贫困发生率仍然很高。

表15：安康市各区县贫困发生率统计

总之，脱贫不仅仅意味着收入水平的上升，同时还要求配套的公共服务的完善。在满足基本的温饱需要后，安康的副业与工业规模都极为有限，呈现出一种与庞大人口失衡的状态。这种经济上的薄弱使得政府的财政能力也较为薄弱，缺乏资金来进行基建投资而改变。因此上级政府的财政支持尤为重要，当地政府通过中央政府拨款、省政府拨款、国有银行的融资、世界银行贷款等多个项目来获得资金。

自从2000年起，当地政府便开始规划对一些贫困村进行基建，具体有修筑道路、通电、通水等，以期为经济发展创造必要环境。另一方面，对于一些建设成本较高的贫困村，政府则采取移民搬迁的做法。总之，政府在这一时期的主导作用非常明显。

2015-2020年的脱贫实践 #

2015年改变统计口径，按户口所在地分划。同一年精准扶贫在全国范围内开始推进，2020年要求全国范围内都要实现脱贫摘帽。安康市也制定了自己的发展规划。

  • 2016年， 全市减贫12.2万人、 脱贫200个村；
  • 2017年， 全市减贫14.3万人、 脱贫260个村；
  • 2018年， 全市减贫15.3万人、 脱贫290个村， 并实现宁陕、 镇坪、 旬阳3个县“摘帽” ；
  • 2019年， 全市减贫16.37万人、 脱贫253个村 。
  • 2020年为全市脱贫攻坚成果巩固年。 至2020年年底， 整体实现扶贫对象不愁吃、 不愁穿， 义务教育、 基本医疗和安全住房有保障的“两不愁三保障” 。

新社区工厂与劳动力外流 #

“新社区工厂”这一概念是在移民搬迁过程中产生的。为了解决新移民的就业问题，当地政府一是提供“退耕还林”、而是在新建设的定居点附近引入企业。但这些企业工资有限，在劳动力自由流动的社会环境下，对青壮年劳动力并不具备吸引力。往往为了多赚钱，男性劳动力都会外出打工。女性由于更多地需要承担家庭责任，因而外出情况较男性为少。但仍有很多女性随丈夫一起外出打工。

毛绒玩具产业的落地与发展 #

毛绒玩具产业落地是在产业迁移的大背景下发生的。改革开放以来江苏省已经在这方面取得了很大的成果。在中央政府的协调下，1991年，苏陕互派干部，希望加强两地间的沟通，学习东部发展经验，加快西部建设。该地分别于1993年和1996年与苏州以及常州签订了合作协议，两地领导干部往来也渐趋紧密。

2017年该地正式从常州引进该产业。18年至21年间，安康市政府及各县区政府推出了一系列毛绒玩具产业政策，主要集中于金融支持、培训补贴、岗位补助、发展贸易企业、发展电子商务、推进社区工厂建设等方面。截止2022年3月，全市已建成毛绒玩具新社区工厂716家，全产业链生产能力初步形成，每天有80万只毛绒玩具由安康发往国内外。

调研行程：

  • 7月19日：汉滨区人社局、工厂参观访谈。

  • 7月20日：紫阳县人社局、工厂参观访谈。

  • 7月21日：汉阴县人社局、工厂参观访谈。

这项产业的直接目的是解决当地妇女的就业。根据我在访谈过程中的记录，车间中各个环节全部是女性在操作。这些工人大部分就是附近的居民，有些是搬迁过来但也有些未搬迁。工厂所开工资并不高而且工作很辛苦，总的来说仍然是一个劳动密集型的产业。

对安康市脱贫实践的一个解释 #

以上部分是基于安康市1949年以来经济发展与脱贫实践，下面根据经济地理方面的知识对当地政府的脱贫措施进行简要解释。安康市处于大山腹地，在农业社会中缺乏经济发展的各种有利条件。根据黄宗智（Huang, 1990）的研究，从明代开始中国经济就进入了漫长的“内卷化”历程。这一过程中，中国始终无法走出马尔萨斯困境，大部分人的生活水平始终在温饱线附近徘徊。直到1978年改革开放以后，市场经济逐渐形成，沿海地区民间的个人企业或村办企业的兴起，成为了中国经济腾飞的开始，由此大幅提高了居民收入，中国沿海地区才率先走出了“内卷化”的困境。这一过程也逐渐由沿海地区向内陆传递，带动了内陆地区的去“内卷化”过程。

陕西省安康地处崇山峻岭间，在内部没有经济发展动力的前提下，实现现代化所以来的只有依靠外来产业与技术的引进。而相比其他长江沿岸城市，从长江三角洲开始的上述提及传输过程势必要缓慢与困难许多。这也就解释了为何在1978年改革开放后，虽然当地政府也及时喊出来“多项经营”这一在全国范围内被普遍使用的口号，但脱贫速度却并不快。因为技术与产业的传播，需要时间。

前边的段落中我们曾简单的将安康辖区按照海拔分为三类：低海拔、中海拔与高海拔地区。当经济发展的浪潮真正传播到该地时，由于市区与县城多位于交通最方便的低海拔地区，自然也最先收获了改革开放的红利，成为了局部的经济中心。而在中海拔地区，还分布着很多人类聚居点，如果想要让产业转移能够落地这些地区，就需要建立起从局部中心到这些聚居点的交通道路。因而这解释了2000年以来的大规模基础设施建设。而在2015年之后，由于脱贫时间点的临近，脱贫任务十分繁重。现实是仍然存在很多高海拔地区需要联系，还有很多贫困人口居住在高海拔地区。如果继续兴建道路连通，可能时间上成本过高，此外大规模的资金也加重了地方负担。因而政府采取了“扶贫搬迁”做法，即在政府的财政支持下，将高海拔地区的聚居点整体搬迁到局部中心可以辐射到的地理位置较好的地点。如此一来，可快速提高脱贫率，在建立的新社区也实行了与之配套的“新社区工厂”来利用新出现的人力资源。

毛绒玩具产业落地该地时间已经迟至2019年，当地的脱贫攻坚任务已基本完成，因而该产业对于当地脱贫成果贡献并不大。该产业的定位是配合所谓的移民新社区建设——为搬迁而来的新移民提高就业机会。以此尽量利用这些留守妇女的空闲劳动力，提高家庭收入。
